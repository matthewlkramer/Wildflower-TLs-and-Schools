# School Logos Upload Structure

Please organize logos in this folder structure:

```
school-logos/
├── school-short-name/
│   ├── main-square.png
│   ├── flower-only.png
│   └── main-rectangle.png
├── another-school/
│   ├── main-square.png
│   ├── flower-only.png
│   └── main-rectangle.png
```

Example:
```
school-logos/
├── yagrumo/
│   ├── main-square.png
│   ├── flower-only.png
│   └── main-rectangle.png
```

The script will automatically match folder names to school short names and upload the correct files.
